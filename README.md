## Chambitos

Chambitos is a comprehensive home services marketplace built with Next.js and TypeScript, connecting service providers with clients seeking professional assistance for their residential needs.

## Getting Started

This project utilizes Next.js for server-side rendering and TypeScript for robust type checking.

### Prerequisites

- Node.js and pnpm installed on your system.

### Installation

1. Clone the repository:

```sh
git clone https://github.com/quibylix/chambitos
cd chambitos
```

2. Install the dependencies:

```sh
pnpm install
```
